# AI Action 提示词优化总结

## 概述

已成功优化 **20+ 个 AI Action 提示词**，从原本简短的模板扩展为详细、结构化的指导系统。

## 优化目标

- ✅ 增加输出信息量和完整度
- ✅ 提供清晰的结构化框架
- ✅ 给出具体的分析指导和要求
- ✅ 确保每个提示词都有明确的字数要求
- ✅ 添加示例格式和详细说明

## 优化对比示例

### 示例 1：价值澄清 (value-clarification)

**优化前（~50字）：**
```
你是一个深度思考助手，擅长从杂乱的信息中提取核心价值。
### 💎 价值澄清
**核心关注**：[总结]
**潜在洞察**：[深层分析]
**回归建议**：[行动指南]
```

**优化后（~600字）：**
- 明确输出要求：2-3个核心关注点，每个需说明为什么重要
- 详细的洞察分析指导：隐藏模式、与过去经验的连接、盲点等
- 具体可执行的行动建议：分短期/中期/长期，每条说明做什么、为什么、预期效果
- 总字数要求：至少 300 字

### 示例 2：第一性原理 (first-principles)

**优化前（~60字）：**
```
你是一个第一性原理思考者。将输入分解为最基本的真理...
**解构**：...
**基本真理**：...
**重构**：...
```

**优化后（~800字）：**
- 三步框架：深度解构 → 定位基本真理 → 全新重构
- 每步都有详细指导和子要求
- 解构：列出假设、质疑假设、识别变量和约束
- 基本真理：至少3条，需论证为什么是真理
- 重构：展开至少200字，说明与原有认知的区别
- 总字数要求：至少 400 字

### 示例 3：六顶思考帽 (six-thinking-hats)

**优化前（~100字）：**
```
请通过"六顶思考帽"视角分析输入。
**⚪ 白帽 (事实)**：...
**🔴 红帽 (情感)**：...
[其他帽子...]
```

**优化后（~1000字）：**
- 每个帽子都有详细的分析指导
- 白帽：列出5-8个具体事实点
- 红帽：充分表达情感和直觉
- 黑帽：识别4-6个风险，建设性批判
- 黄帽：挖掘4-6个价值和利益
- 绿帽：提出3-5个创意想法
- 蓝帽：综合整合和行动方案
- 每个帽子至少 80 字，总计至少 500 字

## 主要改进点

### 1. 结构化框架
每个提示词都有清晰的分析框架，如：
- 第一性原理：解构 → 基本真理 → 重构
- AAR：预期目标 → 实际结果 → 差异分析 → 经验总结
- 终局思维：终局推演 → 分岔点 → 反向规划 → 风险检验

### 2. 明确的输出要求
- 每个部分的字数要求
- 需要列举的数量（如：至少3-5条）
- 具体的分析维度
- 示例格式

### 3. 详细的分析指导
不再是简单的"[...]"占位符，而是：
- 具体要分析什么
- 从哪些角度分析
- 如何展开论述
- 注意事项和原则

### 4. 推荐类的策略设计
所有推荐类都采用 **3+3 策略**：
- 前3个：深度相关/同频共鸣/思想契合
- 后3个：跨界惊喜/反差冲击/脑洞大开
目的：既深化主题，又打破思维边界

### 5. 反思提问机制
月度回顾加入了 7 个深度反思问题：
- 关于成就、情绪、关系、价值等维度
- 促进用户主动思考，而非被动接受分析

### 6. 多层次分析
从数据到洞察的完整路径：
- 数据统计（客观事实）
- 趋势识别（模式发现）
- 深度洞察（原因分析）
- 行动建议（未来规划）
